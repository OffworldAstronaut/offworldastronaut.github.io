Funções

E se... relacionarmos dois conjuntos?

By OffworldAstronaut

O que é uma função?

Podemos definir uma função como sendo uma relação entre dois conjuntos A e B, onde cada elemento de A está relacionado com um e somente um elemento de B. Por meio dessa definição, podemos extrair dois elementos importantes: o domínio da função (A) e o contradomínio (B).

Por essa definição, é comum imaginarmos uma função como sendo uma "caixa" que recebe uma entrada e retorna uma saída. Para transformar as "entradas" em "saídas", a máquina utiliza alguma "instrução" predefinida, que chamamos de "lei de formação" ou "lei de correspondência": essa lei estabelece a relação entre cada elemento dos dois conjuntos.

De uma maneira mais formal, podemos dizer que uma função entre dois conjuntos é um subconjunto do produto cartesiano entre eles, com esse subconjunto satisfazendo alguma restrição ou propriedade.

Domínio, contradomínio e imagem

O domínio de uma função é o seu "conjunto origem". Todos os elementos desse conjunto devem estar relacionados com apenas um elemento do contradomínio para que a relação entre esses esses dois conjuntos seja chamada de "função".
Na notação matemática, chamamos o domínio de uma certa função f de Dom(f).

O contradomínio é o conjunto "dependente do domínio". Não necessariamente todos os elementos do contradomínio estão relacionados com pelo menos um elemento do domínio, o que nos leva à definição do conjunto imagem. Quando utilizamos a notação, podemos chamar o contradomínio de CD(f).

O conjunto imagem é o conjunto de todos os elementos do contradomínio que possuem uma relação com pelo menos um elemento do domínio.
Ele pode ser "menor" (estar contido) ou igual ao contradomínio, mas nunca "maior" (conter o CD).

Na notação matemática, chamamos o conjunto imagem de Im(f).

Gráfico de uma função

Podemos representar uma função graficamente por meio do plano cartesiano: organizamos os conjuntos nas duas retas perpendiculares entre si, com o eixo horizontal (eixo das abcissas) sendo o conjunto domínio da função e o eixo vertical (eixo das ordenadas) sendo o conjunto contradomínio.

Por conta da configuração do plano, cada ponto representa um par ordenado entre dois conjuntos quaiquer, embora geralmente utilizamos funções com domínio nos Números Reais e contradomínio também nos Reais.

Tipos de funções

No estudo das funções, podemos dividi-las em algumas categorias para facilitar seu estudo mais profundo.

Injetoras

As funções injetoras, também chamadas de "funções um-para-um" (one-to-one) ou ainda funções injetivas são funções em que cada elemento do conjunto imagem está relacionado com apenas um elemento do domínio.

Formalmente, elas são definidas como funções que satisfazem a relação:

Se  f ( x 1 ) = f ( x 2 ) , e n t a ˜ o   x 1 = x 2

Ou seja, se "dois" elementos do conjunto imagem são iguais, então "eles" partem do mesmo elemento do domínio, ou seja, era o mesmo elemento o tempo inteiro!

É mais fácil entender o caso inverso:

Se  f ( x 1 ) ≠ ( x 2 ) , e n t a ˜ o   x 1 ≠ x 2

Ou seja, se os elementos do conjunto imagem são diferentes entre si, os elementos do domínio que os originaram também são diferentes entre si.

Funções sobrejetoras

As funções sobrejetoras ou sobrejetivas são funções em que todo elemento do contradomínio é um elemento do conjunto imagem, ou seja, os conjuntos "contradomínio" e "imagem" são iguais.

Funções bijetoras

As funções bijetoras ou bijetivas são a "fusão" das duas anteriores, funções em que cada elemento do conjunto imagem está relacionado com apenas um elemento do domínio, com os conjuntos imagem e contradomínio sendo iguais.
