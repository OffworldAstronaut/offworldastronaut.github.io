Definição

Antes de tudo... o que é um conjunto?

Sem precisar entrar no campo bizarro das definições formais da matemática, podemos afirmar que um conjunto é um grupo, uma coleção de símbolos ou objetos, que damos o nome de elementos do conjunto.

Conjunto vazio

Para continuar no campo de relações que os mais variados conjuntos podem ter entre si, precisamos definir um conjunto especial, chamado conjunto vazio.

O conjunto vazio, como o nome experime, é vazio, i.e não possui nenhum elemento. Esse conjunto é representado com o símbolo ∅, muitas vezes confundido com um ϕ (phi).

Por conta da propriedade que o define, o conjunto ∅ está sempre contido (⊂) em qualquer outro conjunto A, já que se isso fosse falso, implicaria que existe algum elemento do conjunto ∅ que não pertence a A, mas como ∅ não possui elementos, afirmar isso seria uma contradição, um absurdo.

Relações

Os conjuntos podem possuir diversas relações (como contingência) entre elementos e outros conjuntos.

Conter ou não conter... eis a questão

Quando todos os elementos de um conjunto A pertencem também a um conjunto B, dizemos que o conjunto A está contido no conjunto B, e representamos essa relação com a notação A ⊂ B ("A está contido em B").

De forma análoga, se também for verdade que B ⊂ A ("B está contido em A"), podemos afirmar que A = B: todos os elementos do conjunto A estão no conjunto B e vice-versa.

União e intersecção

Definimos a união de dois conjuntos A e B como sendo o conjunto de todos os elementos que estão em A, ou estão em B, ou seja, seria realmente criar um conjunto que "junta" os dois outros. Em notação, representamos assim:

A ∪ B = { x | x ∈ A ∨ x ∈ B }

De forma parecida, definimos a intersecção de dois conjuntos como sendo o conjunto formado por todos os elementos que estão em A e em B ao mesmo tempo. Representamos por meio da notação desse jeito aqui:

A ∩ B = { x | x ∈ A ∧ x ∈ B }

Diferença e complementares

A diferença entre dois conjuntos A e B é definida como sendo o conjunto de todos os elementos que estão em A e não estão em B. Representamos essa relação pela notação desta maneira:

A ∖ B = A − B = { x | x ∈ A ∧ x ∉ B }

Assim, podemos dizer que o "conjunto complementar de" B com relação a A é A - B.

Produto Cartesiano

Definimos produto cartesiano como sendo a união dos pares ordenados, onde a primeira entrada é um elemento do conjunto A e a segunda entrada é um elemento do conjunto B.

Par ordenado é um par de elementos, com o primeiro pertencente ao primeiro conjunto e o outro pertencente ao segundo.

Matematicamente, temos a seguinte definição:

A × B = { ( a , b ) | a ∈ A , b ∈ B }
